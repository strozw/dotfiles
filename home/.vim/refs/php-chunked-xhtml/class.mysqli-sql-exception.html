mysqli_sql_exception クラス

(PHP 5)

導入

mysqli の例外を処理するクラスです。

クラス概要

mysqli_sql_exception extends RuntimeException {
/* プロパティ */
protected string $sqlstate ;
/* 継承したプロパティ */
protected string $message ;
protected int $code ;
protected string $file ;
protected int $line ;
}

プロパティ

sqlstate

エラーの SQLSTATE。
