Stackable クラス

(PECL pthreads >= 0.36)

導入

Stackable は、ワーカースレッドが実行するタスクです。 Stackable オブジェクトの読み書きは、その実行前や実行後あるいは実行中に同期できます。

クラス概要

Stackable {
/* メソッド */
final public boolean isRunning ( void )
final public boolean isTerminated ( void )
final public boolean isWaiting ( void )
final public boolean lock ( void )
final public boolean notify ( void )
abstract public void run ( void )
final public mixed synchronized ( Closure $block [, mixed $... ] )
final public boolean unlock ( void )
final public boolean wait ([ string $timeout ] )
}

目次

  • Stackable::isRunning — 状態を検出する
  • Stackable::isTerminated — State Detection
  • Stackable::isWaiting — 状態を検出する
  • Stackable::lock — 同期処理
  • Stackable::notify — 同期処理
  • Stackable::run — 実行する
  • Stackable::synchronized — 同期処理
  • Stackable::unlock — 同期処理
  • Stackable::wait — 同期処理
