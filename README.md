

## Setup

1. install rcm  

  ```
  brew install rcm
  ```

2. clone repository  

  ```
  cd ~/
  git clone https://github.com/strozw/dotfiles .dotfiles
  ```

3. rcup  

  ```
  rcup
  ```

## Add other dotfile

* For file  

  ```
  cd ~/
  mkrc .vimrc
  ```

* For directory  

  ```
  cd ~/
  mkrc -d .vim
  ```
