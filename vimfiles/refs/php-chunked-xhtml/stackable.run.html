Stackable::run

(PECL pthreads >= 0.36)

Stackable::run — 実行する

説明

abstract public void Stackable::run ( void )

このメソッドはワーカースレッドが実行します。

パラメータ

この関数にはパラメータはありません。

返り値

このメソッドの返り値は無視されます。
