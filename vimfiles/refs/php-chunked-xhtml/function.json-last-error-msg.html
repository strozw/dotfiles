json_last_error_msg

(PHP 5 >= 5.5.0)

json_last_error_msg — 直近の json_encode() や json_decode() の呼び出しのエラー文字列を返すl

説明

string json_last_error_msg ( void )

パラメータ

この関数にはパラメータはありません。

返り値

成功した場合にエラーメッセージを返します。パラメータが間違っている場合は NULL を返します。
